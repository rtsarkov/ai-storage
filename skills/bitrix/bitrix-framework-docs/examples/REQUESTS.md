# Request Examples

Use this skill for requests like:

- "Проверь контроллер Bitrix D7: не отключили ли CSRF и правильно ли возвращается компонент."
- "Сделай DataManager для новой таблицы и подскажи, где нужна миграция."
- "Оптимизируй выборку списка: сейчас N+1 по связанным сущностям."
- "Почему `LIMIT` ломает выборку коллекции с множественными свойствами?"
- "Как правильно подключить JS-расширение через Bitrix Extension?"
- "Сверь решение с официальной документацией Bitrix и локальной версией ядра."

Expected output:

- выбранный Bitrix API и причина выбора;
- конкретные изменения или review findings;
- риски совместимости с локальным ядром;
- проверки: syntax, endpoint/manual scenario, cache invalidation, SQL/query count.
